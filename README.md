### README

Ubuntu 기준으로 작성하였습니다.

```
client는 Vue.js로 동작하고 server는 Node.js로 동작합니다. 
server를 먼저 켜신 후에 client를 실행하면 Todo 앱을 실행하실 수 있습니다.
```



##### Requirements

* Node.js 설치

  * 저장소 추가

    ```
    curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
    ```

  * 설치

    ```
    apt-get install nodejs
    ```
